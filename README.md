# Projeto-WikiLitica
Projeto para a matéria de Fundamentos de Programação Internet de HTML/CSS.

Notas ao Usuário
-----------------
-Usuário Adm para utilizar todo o sistema de cadastro, menos cadastrar candidato: usuário: admin senha:admin. Para conseguir a permissão acesse a página templates/admin.php

Funcionalidades do Site
------------------------------------------------------
-O site possui um sistema de cadastro e login do usuário;
-Os usuários cadastrados podem cadastrar políticos depedendo da sua permissão de acesso;
-Qualquer usuário pode realizar uma busca filtrando por estado e função;
-Os usuários cadastrados pode editar e excluir os canditados buscados;
-O usuário tem acesso ao histórico de partidos e cargos ocupados do político;
